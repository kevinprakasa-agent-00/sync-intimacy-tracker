# Assets Required

## App Icons

Create or download app icons and place them here:

- `icon.png` - Main app icon (1024x1024)
- `adaptive-icon.png` - Android adaptive icon (1024x1024) 
- `splash.png` - Splash screen (1242x2436)
- `favicon.png` - Web favicon (32x32)

## Textures

The app uses a paper texture background:

- `paper-bg.png` - Subtle paper texture for backgrounds

You can create one using:
- https://www.transparenttextures.com/ (search for "paper")
- Or any subtle cream/off-white paper texture

## Recommended App Icon Design

The icon should reflect the app's soft, romantic aesthetic:
- Soft pink/peach background (#FFF5F0 or #FFB6C1)
- Simple heart or interlocking hearts in the center
- Rounded corners matching the app's design language
- No harsh shadows or gradients
